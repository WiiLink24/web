My Avatar Editor

You can use this tool called My Avatar Editor to make and edit Miis. It runs on Adobe Flash, which was discontinued at the end of 2020. However, there are still a few ways to use it.
Check out mii.rc24.xyz for more Mii content!

Install for Desktop

It's possible to install My Avatar Editor on your computer as an application. Follow these instructions:
  1. Download and install Adobe AIR. ( macOS / Windows) AIR is discontinued by Adobe, but it's still possible to use it.
  2. Download MyAvatarEditor.air, but don't open it yet.
  3. Set your computer's clock sometime before October 31, 2017. If you don't do this step, Adobe AIR will say the installer file is damaged.
  4. Open MyAvatarEditor.air and install it.
  5. Set your computer's clock to the latest time now.

You can now open My Avatar Editor at any time. Miis with the file extension ".mae" will now be associated with the application.

Flashpoint

You can use BlueMaxima's Flashpoint to use My Avatar Editor again. It's a project to preserve old Flash games. For more information, please see the official Flashpoint website.

Use in Browser

This won't load nor show up unless you're somehow using a legacy browser. (It will work with SeaMonkey)
